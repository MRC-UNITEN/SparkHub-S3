# 💡 MRC SparkHub: WS2812B Addressable LED Workshop

Welcome to the **Mobile Robotics Club (MRC) SparkHub Series**! This repository contains all the code examples and logic used in our hands-on workshop on WS2812B RGB LED strips. 

Whether you are from Engineering, CS, IT, Business, or Accounting, these modules are designed to take you from lighting your first LED to creating complex, professional light animations using the **FastLED** library.

---

## 🛠 Hardware Requirements
* **Microcontroller:** Arduino Uno (or compatible)
* **LED Strip:** WS2812B (10 LEDs used in these examples)
* **Power:** USB Power (Safe limits set to 500mA in code)

## 💻 Software Setup
1. Install the **Arduino IDE**.
2. Open the Library Manager (**Ctrl+Shift+I**) and search for `FastLED`.
3. Install the latest version.

---

## 📂 Workshop Modules

Each folder corresponds to a specific concept we covered during the session:

| # | Folder | Concept Taught |
|---|---|---|
| 1 | `FirstLED` | The basics: Wiring, Power safety, and lighting LED `[0]`. |
| 2 | `Array` | Introduction to C++ Arrays and `for` loops. |
| 3 | `functions` | Using built-in FastLED shortcuts like `fill_rainbow`. |
| 4 | `Colors` | Understanding Hex codes, RGB, and the HSV system. |
| 5 | `Color pallete` | Working with pre-designed "Brand" color themes. |
| 6 | `beatpulse` | Using Sine waves for "breathing" light effects. |
| 7 | `twinkle` | Randomness and probability (Glitter effect). |
| 8 | `comet` | Persistence of vision and "fading" tail effects. |
| 9 | `mirror` | Mathematical symmetry (Mirroring effect). |
| 10 | `colorblend` | Smoothly transitioning between colors. |
| 11 | `bounce` | Bounce end to end |

---

## 🚀 How to Use
1. Clone this repository.
2. Open the `.ino` file inside any numbered folder.
3. Ensure your `DATA_PIN` is correct.
4. Upload and enjoy the light show!
